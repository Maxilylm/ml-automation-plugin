# Report Generation

## When to Use
- Creating EDA summary reports
- Generating model performance reports
- Building drift monitoring dashboards
- Summarizing project status

## Report Types

- **EDA Report**: Data quality, distributions, correlations
- **Performance Report**: Model metrics, comparisons, trends
- **Drift Report**: Feature drift, prediction drift monitoring
- **Status Report**: Project progress, milestones, next steps

## Workflow

1. Identify report type and audience
2. Gather relevant data and metrics
3. Generate visualizations
4. Write narrative insights
5. Format and deliver report
