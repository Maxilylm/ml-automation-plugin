# Testing

## When to Use
- Generating tests for Python modules
- Validating test coverage meets thresholds
- Running unit and integration test suites

## Workflow

1. Analyze target modules
2. Generate unit tests for functions and classes
3. Generate integration tests for workflows
4. Run tests with pytest
5. Validate coverage meets 80% threshold
6. Report results and coverage gaps
